## Description

This project is a recreation of the classic Pong game built using Raylib in C++.

The game features:

- Two-player mode (W/S and Up/Down controls)
- Optional single-player mode with AI opponent
- Multiple ball mode (1–3 balls)
- Score tracking
- Win screen when a player reaches 3 points
- Scaled textures and background rendering
- Delta-time based movement for smooth gameplay

---

## Controls

| Key | Action |
|-----|--------|
| W / S | Move Player 1 |
| Up / Down | Move Player 2 |
| T | Toggle Single Player Mode |
| 1 | 1 Ball |
| 2 | 2 Balls |
| 3 | 3 Balls |
| Close Window | Exit Game |

---

## Features Implemented

- Delta time for all movement
- Manual AABB collision detection
- Dynamic number of balls
- AI opponent logic (tracks ball position)
- Scaled textures using `DrawTexturePro`
- Background scaling to window size
